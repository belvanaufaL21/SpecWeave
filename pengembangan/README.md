# Pengembangan

Folder ini berisi tools dan konfigurasi untuk development.

## Struktur Folder

### .kiro/
Konfigurasi untuk Kiro AI assistant:
- `specs/` - Spesifikasi fitur dan requirements
- `steering/` - Steering files untuk AI guidance
- `settings/` - Settings dan konfigurasi

### .vscode/
Konfigurasi Visual Studio Code:
- `settings.json` - Workspace settings
- `extensions.json` - Recommended extensions
- `launch.json` - Debug configurations

## Spesifikasi Fitur

### Interactive METEOR Testing
- Requirements, design, dan tasks untuk fitur testing METEOR
- PDF dokumentasi METEOR

### SpecWeave Enhancement
- Requirements dan tasks untuk enhancement aplikasi
- Dokumentasi teknis dan spesifikasi

## Development Workflow

1. Baca spesifikasi di `.kiro/specs/`
2. Setup development environment
3. Gunakan VS Code dengan recommended extensions
4. Follow steering guidelines untuk AI assistance

## AI Assistant Integration

Kiro AI assistant dikonfigurasi untuk:
- Memahami struktur proyek
- Mengikuti best practices
- Memberikan guidance sesuai spesifikasi
- Membantu debugging dan development
